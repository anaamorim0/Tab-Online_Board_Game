# Tâb — Jogo de Tabuleiro

Este projeto consiste na implementação do jogo de tabuleiro Tâb.
O trabalho foi desenvolvido em duas partes:
**Parte 1 — Jogador vs IA (Modo Local)**
O objetivo é implementar toda a lógica do tabuleiro, o movimento das peças e a interação com o jogador, permitindo partidas contra um adversário artificial controlado por um algoritmo de decisão.

**Fase 2 — Jogador vs Jogador (Modo Online)**
O objetivo é permitir que dois jogadores joguem entre si em computadores diferentes, através de comunicação via servidor, mantendo a mesma lógica do jogo, mas sincronizando os estados entre clientes.
